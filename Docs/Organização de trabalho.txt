Projeto FrontEnd - Organização de trabalho:

Planeamento:
- definição dos objetivos
- definição dos componentes visualizados em cada momento
- definição das tecnologias necessarias para o funcionamento
- esboço da documentação genérica
- definir prazos

Desenvolvimento:
- organização inicial do projeto por componentes mais básicos, por layouts (grupo de componentes no mesmo layout), por views (grupo de layouts visiveis numa pagina)
- desenvolvimento da parte visual dos componentes
- implementação do funcionamento dos componentes
(a decidir se vai ser feito para cada componente ou para a aplicação)

Git:
- Usar branch dev para o desenvolvimento
- Usar um branch do dev por cada componente
- Fazer pequenos commits e fazer sempre que acabe o html, o css e o javascript
- No fim de cada compomente, teste, merge para dev, teste geral, escrever uma parte da documentação, push

Final:
- Build e deploy
- auto avaliação
